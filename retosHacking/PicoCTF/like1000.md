- like1000
#### Description

This [.tar file](https://challenge-files.picoctf.net/c_fickle_tempest/96ad54735d25c18a159eb22cd408adc8dad73f855113b1b700a769d4fa9f2c10/1000.tar) got tarred a lot.
## solucion
- Verificamos que tipo de archivo es :

`file 1000.tar`

- Vemos el contenido sin descomprimir, vemos que es un multiple tar con varias capas, presumiblemente 1000

`tar -tf 1000.tar`

- descomprimimos y vamos borrando los archivos anteriores

`for i in {1000..1}; do tar -xf $i.tar && rm $i.tar ; done`

- abrimos la bandera

`open flag.png`
